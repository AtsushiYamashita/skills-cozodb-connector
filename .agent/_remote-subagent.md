- リモートサブエージェント（実験的） | Gemini CLI
  - https://geminicli.com/docs/core/remote-agents/

/agents list: 使用可能なすべてのローカルおよびリモート サブエージェントを表示します。
/agents refresh: エージェントレジストリを再読み込みします。エージェント定義ファイルを追加または変更した後に使用してください。
/agents enable <agent_name>: 特定のサブエージェントを有効にします。
/agents disable <agent_name>: 特定のサブエージェントを無効にします。
